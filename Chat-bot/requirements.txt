# Project Requirements and Installation Guide

This document explains the required packages and how to install and run the chatbot on Windows and Unix-based systems.

---

## Required Files

Ensure your project directory contains the following files:

```
chatbot.py
requirements.txt
.env
.gitignore
``
## Required Packages

mistralai
python-dotenv

---

## Environment Variable Setup

Create a file named `.env` in the project directory and add the following line:

```
API_KEY=your_mistral_api_key_here
```

Replace `your_mistral_api_key_here` with your actual API key.

---

# Installation Instructions

## 1. Install Python

Make sure Python 3.9 or later is installed.

Check version:

```
python --version
```

or

```
python3 --version
```

If Python is not installed, download it from:
[https://www.python.org/downloads/](https://www.python.org/downloads/)

---

## 2. Create a Virtual Environment (Recommended)

### Windows

```
python -m venv venv
venv\Scripts\activate
```

### macOS / Linux / Unix

```
python3 -m venv venv
source venv/bin/activate
```

---

## 3. Install Project Requirements

### Windows

```
pip install -r requirements.txt
```

### macOS / Linux / Unix

```
pip3 install -r requirements.txt
```

---

# Running the Chatbot

### Windows

```
python main.py
```

### macOS / Linux / Unix

```
python3 main.py
```

When prompted, type your input and press Enter.
To exit the program, type:

```
exit
```

or

```
quit
```

---

# Common Issues

1. ModuleNotFoundError: No module named 'dotenv'

Install:

```
pip install python-dotenv
```

2. ModuleNotFoundError: No module named 'mistralai'

Install:

```
pip install mistralai
```

3. API key not loading

* Ensure `.env` exists in the same folder as `main.py`
* Confirm the line format in `.env`:

  ```
  API_KEY=your_key_here
  ```
* Verify that `.env` is listed in `.gitignore`

---

This completes the setup and installation process.
You can now successfully run the chatbot in your terminal.
